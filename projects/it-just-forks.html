React

The project was realized using the React framework. In the code structure, I utilized a range of modern features, including functional components and hooks. I used the react router to navigate between different parts of the app.

Tailwind

The visual side of the project was achieved with the use of Tailwind CSS. I find it’s utility-based system to be very flexible and convenient to use. I used Tailwind’s responsiveness model to create an app that would scale to any device.
