Ungdomsringens Musikfestival

Description

Ungdomsringen realizes a yearly event called Ungdomsringens Musikfestival – the largest youth festival in the Nordic region. The problem the festival faced was the lack of a dedicated website. Our task was to create a responsive website form Ungdomsingens Musikfestival, that would serve both the promotional and informatory role.

My roles:

  • Main JavaScript programmer

  • Designer

View on GitHub

View live

Process summary

By carrying out thorough research and analysis, as well as performing relevant benchmarking, we were able to shape an idea of what the end product should look like. User interviews helped set us in the right direction, as well as analyzing data and existing large-scale usability studies.

We created an appealing visual identity for the festival to strengthen its’ position among competitors. The solution was implemented using HTML, CSS and Javascript, along with Wordpress, used as a headless CMS.

Design

With the website being targeted at teenagers, and most teenagers spending their time on their phones, we followed a mobile-first design scheme.

I was the lead designer, performing regular quality checks on our work. I was responsible for combining the ideas of other team members into a cohesive, unique prototype that could easily be altered through the use of a CMS. The end result of the design phase was a high fidelity mock-up.

CMS integration

We used Wordpress as CMS, utilizing it’s REST API to display content on the website. I was in charge of coding the http requests to the API. I utilized my knowledge of JavaScript to parse the responses and elegantly display them on the website. I was working with AJAX to create the http request. To first familiarize myself with the API structure, I used Postman to perform test requests.
